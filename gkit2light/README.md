## Compilation
Dans le répertoire gkit2light/, effectuez les commandes ci-dessous :
```
cmake .
```  
puis
```
make
```  
pour compiler le projet.

## Execution
Toujours dans le répertoire gkit2light/, effectuez la commande :
```
./bin/MDS2019.exe
```
pour lancer le jeu.
